# 💬 AI Chatbot using Transformers (Streamlit + Hugging Face)

## 🚀 Overview

This project is a **web-based AI chatbot** built using **Streamlit** and powered by transformer-based language models via the Hugging Face Inference API.

It provides an interactive chat interface similar to modern AI assistants, allowing users to have real-time conversations with an AI model. The application maintains chat history, supports dynamic responses, and offers a smooth user experience with features like typing animations and chat reset functionality.


## 🎯 Purpose of the Project

This project was built as a **hands-on learning experience to understand and work with Transformer-based models**.

Through this project, I explored:

* How transformer models work in real-world applications
* How to integrate Hugging Face Inference APIs
* Managing conversational context (chat memory)
* Building interactive AI interfaces using Streamlit
* Structuring prompts and system instructions for better AI responses

The main goal was not just to build a chatbot, but to **gain practical knowledge of deploying and interacting with transformer models in an application environment**.
